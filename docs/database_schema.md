## 爱在欧宝APP 数据库设计
https://juejin.cn/post/6844903833668190216
### 1.用户信息表

| user_info   |      |                     |
| :---------- | ---- | ------------------- |
| 字段        | 类型 | 说明                |
| id          |      | 唯一标识（主键）    |
| nick_name   |      | 宝宝昵称            |
| gender      |      | 性别                |
| month_age   |      | 年龄（月）          |
| register_time |      | 注册时间            |
| level       |      | 等级                |
| jifen       |      | 积分                |
| phone_num   |      | 电话号码-登录用户名 |
| Password    |      | 登录密码            |
|             |      |                     |

### 2. 游戏资源（按照月龄和类别tag查询）

| game_resources |      |                              |
| -------------- | ---- | ---------------------------- |
| 字段           | 类型 | 说明                         |
| id             |      | 主键                         |
| title          |      | 游戏名称                     |
| description    |      | 游戏文字描述                 |
| belong_age     |      | 适合月龄（0-3岁分成6个区间） |
| tag            |      | 游戏类别标签                 |
| video          |      | 视频展示                     |
| picture        |      | 封面图片                     |
|                |      |                              |



### 2.育儿推文表（按月龄和tag 查询）

| article_resources |      | 包括温州宝宝模块，里面用tag筛选 |
| ----------------- | ---- | ------------------------------- |
| 字段              | 类型 | 说明                            |
| id                |      | 主键                            |
| title             |      | 文章标题                        |
| content           |      | 文章正文                        |
| tag               |      | 文章标签（自定义）              |
| belong_age        |      | 适合月龄（0-3岁分成6个区间）    |

### 3. 贝贝日记表

| 字段       | 类型 | 说明                  |
| ---------- | ---- | --------------------- |
| author     |      | 发帖用户（外键 user） |
| post_text  |      | 帖子中的文字          |
| post_image |      | 帖子中的图片          |
| post_date  |      | 发帖时间              |
|            |      |                       |

### 4.测评试题表

| 字段     | 类型 | 说明     |
| -------- | ---- | -------- |
| question |      | 问题     |
| tag      |      | 试题类别 |
| content  |      | 内容     |
| answer   |      | 答案     |
